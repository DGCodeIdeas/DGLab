# Phase 4: Analytics & User Engagement

## 4.1 PWA Mode Analysis
- **Standalone vs. Browser**: Tracking how many users are launching the app from their home screen (PWA) vs. visiting via a web browser.
- **Session Duration**: Comparative analysis of engagement time between PWA and browser users.
- **Retention Rates**: Tracking the long-term usage patterns of users who "Install" the PWA.

## 4.2 Performance Monitoring (Core Web Vitals)
- **LCP, FID, CLS**: Real-time monitoring of Core Web Vitals to ensure a smooth user experience.
- **Asset Load Times**: Tracking the performance of the `AssetService` in serving compiled JS/CSS.
- **Time to Interactive (TTI)**: Specifically for the PWA boot sequence.

## 4.3 Device Distribution
- **Browser/OS Breakdown**: Understanding the environment where users are running the PWA.
- **Screen Sizes**: Optimizing UI/UX based on actual device dimensions.
- **Geolocation (Optional)**: Privacy-preserving region-level analytics to optimize CDN/Server locations.
